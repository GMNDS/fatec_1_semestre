Evolução do Software

Uma linha do tempo de como o sofwar evoluiu.

    Linha do tempo

  • Assembly (1949) - Assembly surgiu no final da década de 40, como a segunda geração de linguagem de programação (anteriormente havia código de maquina) e era uma forma mais legível para humanos programarem na época. É por volta dessa época também que as linguagens de computadores começam ser usadas de forma comercial. Nessa época os computadores eram programados através de válvulas
  • UNIVAC I (1951) - O primeiro computador comercialmente diponível. Foi utilizado para fins militares, comerciais e de pesquisa.
  • IBM (1952) - A IBM entrou no mercado com o IBM 701 que ra voltado para aplicações ciêntificas em 1952. Ele deu origem a diversas outras máquinas como IBM 704, 709 e 7094, tornando a IBM a maior fabricante de computadores da época.
  • Linguagens de alto nível (1954-1960) - Depois do primeiro computador transitorizado ter cido construído em 1954 começaram surgir linguagens de mais alto nível como FORTRAN(1956), ALGOL 58(1957), COBOL(1959) e a evolução do algol o ALGOL 60(1960).
  • Multiprogramação (1965) - A multiprogramação surgiu com a evolução do hardware uando o uso de circuitos integrados ao invés de transistores separados começaram ser utilizados. Dessa forma era possível executar programas de forma alternada sem deixar a CPU ociosa, porém haviam problemas graves de segurança que foram sendo corrigidos com desenvolvimento de novos Sistemas operacionais. Por volta dessa época também começaram a surgir as softwares houses.
  • Termo "Engenharia de Software" (1968) - O termo é utilizado desde o começo da década de 60, mas somente em 1968 foi mencionado oficialmente em uam conferência da OTAN sobre Engenharia de software "Jnato Conference On Software Engineering".
  • Crise do sofware (1969) - A crise do sofware começou no final da década de 60 quando a engenharia de software ainda estava no ínicio, os sistemas começaram ficar mais complexos, os requisitos não eram bem definidos, os códigos não eram documentados e não haviam estruturas a serem seguidas para a facil manutenibilidade, oque causava softwares que nunca eram terminados ou eram finalizados com 150 a 400% do seu orçamento e prazos elevados. O termo se popularizou na década de 1970
  • Banco de dados relacional (1970) - Em 1970 Ted Codd, publicou o primeiro artigo sobre banco de dados relacionais. Oque levou a IBM a montar um grupo de pesquisa conhecido como Sytem R que posteriormente criou o SQL/DS que usava linguagem SQL para consultas.
  • Metodologias de Sotware (1970) - Os primeiros modelos formais da engenharia de software foram desenvolvidos. O principal sendo o modelo cascata que consiste em fases sequenciais para a criação do software(requisitos, design, implementação, teste e manutenção)
  • Linguagem C (1972) - A criação da linguagem C (derivada de B, que era derivada de ALGOL) foi extremamente importante para engenharia de software. Se tornou a linguagem mais utilizado para o desenvolvimento de sistemas operacionais e software, permitindo aos desenvolvedores desenvolverem código eficiente e portável entre diferentes plataformas.
  • MS-DOS E GNU (1983) - A microsoft lançou o MS-DOS(Microsoft Disk Operating System) junto com um acordo com a IBM, revolucionando o mercado de software da época já que antes os computadores eram vendidos sem um SO e os usuários precisavam comprar ou desenvolver seus próprios sistemas. Nesse mesmo ano foi desenvolvido o GNU, um movimento criado por Richard Stallman para desenvolver um sistema operacional completo de software livre e código aberto que permitiria os usuários estudarem, modificarem e distribuir o código fonte.
  • ISO/IEC 12207 (1986) - Essa norma criada no final da décadada de 80 fornece um conjunto de práticas e diretrizes para o ciclo de vida de desenvolvimento de software, oque trouxe diversos benefícios como padronização de processos e práticas, que facilitava a colaboração entre equipes e melhorou a qualidade dos produtos finais. Além de estabelecer uma base comum para avaliação e melhoria no processo de desenvolvimento.
  • Kenel Linux e Python (1991) - Linus Torvalds criou e lançou o kernel Linux como um projeto de código aberto, impulsionando o movimento de software livre. Esse se tornou um dos sistemas mais populares do mundo e o mais utilizado em servidores. Nesse mesmo ano Guido van Rossum criou Python que hoje é a linguagens mais popular do mundo impulsionado pelas suas bibliotecas de IA.
  • Manifesto Ágil (1994) - No meio da década de 90 foi criado o manisfeto ágil, que enfatiza a entrega de software em ciclos curtos e colaboração com clientes. Teve grande impacto na engenharia de software levando surgimentos de metodologias como o Scrum.
  • Java (1995) - A Sun Microsystems lançou a linguagem de programação Java, que se tornou amplamente utilizado em todo o mundo, pois, permitia que programas funcionassem em diversas plataformas sem recompilações. Oque levou a surgir aplicações das mais diversas, mobile, desktop, web, etc.
  • Era da computação em Nuvem e Automatação(2000) - A partir do século 21 a engenharia de software começou ser cada vez mais influencida pela computação em nuvem, com isso também surgiu necessidade de enviar essas aplicações para o nuvem com cobertura de testes e de forma automatizada.
  • Devops e IA(2010) - A partir de 2010 o movimento devops que visa integração e entrega contínua, além de várias boas práticas para estruturar software ganhou mais força visando a colaboração e automatação para acelerar o processo de desenvolvimento. O aprendizado de maquina apesar de já existir também ganha força, impulsionada por avanços no hardware e disponibilidade de grandes conjuntos de dados.
  • Tensor Flow E Visual Studio Code (2015) A microsoft lança o VSCODE com seu núcleo de código aberto, um editor de código multiplataforma que é amplamente utilizado nos dias de hoje. Também foi criado o TensorFlow um framework de aprendizado de máquina que tornou mais fácil para os desenvolvedores criarem aplicações com recursos de inteligência artificial.
  • Atualmente (2024) - A engenharia de sofware continua em constante evolução.
