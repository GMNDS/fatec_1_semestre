# Bash

## Operadores lógicos

- Igual: `Equal` ou `==`
- Diferente: `NEQ`
- Menor que: `LSS`
- Menor ou igual a: `LEQ`
- Maior que: `GTR`
- Maior ou igual a: `GEQ`
- Condição negativa: `NOT`
- Existe: `EXIST`
- Contador `set /a cont=%cont% + 1`
- Números randômicos:`set /a numero=(%Random% %% 10) + 1`
- Estrutura de repetição: `/L %%n in (1,1,4) do (Estrutura de repetição)`
